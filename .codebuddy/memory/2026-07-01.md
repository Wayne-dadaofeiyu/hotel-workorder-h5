## InRoomBadge 图标优化
- 从第一性原理分析：图标并行处理 > 文字串行阅读，形状+颜色双编码
- 迭代过程：
  1. 文字 "Occupied/Unmanned" → User/UserX 图标（16px）— 用户要求放大
  2. 放大到 20px — 用户觉得不够明显
  3. 加圆形底座(30x30px) + ring 外圈 — 用户满意底座但 UserX 形态问题
  4. **最终方案**：UserRound (人在) + DoorOpen (人不在) + 圆形底座 + ring 外圈
- 关键决策：DoorOpen 替代 UserX，解决 "只多了个 X 不协调、绿色高亮时 X 不明显" 的问题
- 颜色语义保持不变：delivery 人在=绿 / cleaning 人不在=绿
- 修改文件：`src/components/common/InRoomBadge.tsx`
